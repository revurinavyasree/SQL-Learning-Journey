# Day 1 - What is SQL?

## What is SQL?

SQL (Structured Query Language) is a standard language used to communicate with relational databases.

It allows users to:

* Store data
* Retrieve data
* Update data
* Delete data

## Why SQL?

Almost every modern application stores data in a database. SQL helps applications interact with databases efficiently.

## Real-World Applications

* Banking Systems
* E-Commerce Websites
* Hospital Management Systsems
* College Management Systems
* Social Media Platforms

## Advantages of SQL

* Easy to learn
* Fast data retrieval
* Standardized language
* Supports large amounts of data

## Common Databases Using SQL

* MySQL
* PostgreSQL
* Oracle Database
* Microsoft SQL Server

## Summary

SQL is the foundation of database management and is one of the most important skills for software engineers and data professionals.
