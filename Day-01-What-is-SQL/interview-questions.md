# Day 1 Interview Questions and Answers

## 1. What is SQL?

SQL is a language used to communicate with relational databases. It helps users store, retrieve, update, and delete data.

## 2. What does SQL stand for?

SQL stands for Structured Query Language.

## 3. Why is SQL used?

SQL is used to manage and manipulate data stored in relational databases efficiently.

## 4. What is a database?

A database is an organized collection of related data that can be stored, managed, and retrieved easily.

## 5. What is the difference between data and information?

Data refers to raw facts and figures, while information is processed data that has meaning and can be used for decision-making.

Example:

* Data: 85, 90, 78
* Information: Student scored an average of 84.3 marks.

## 6. Name some popular SQL databases.

* MySQL
* PostgreSQL
* Oracle Database
* Microsoft SQL Server

## 7. What operations can be performed using SQL?

Using SQL, we can:

* Insert data
* Retrieve data
* Update data
* Delete data
* Create tables and databases

## 8. Is SQL a programming language?

SQL is generally considered a database query language rather than a full programming language.

## 9. What are the advantages of SQL?

* Easy to learn
* Fast data retrieval
* Standardized language
* Efficient data management
* Widely used in industry

## 10. Where is SQL used in real-world applications?

SQL is used in:

* Banking Systems
* School Management Systems
* Hospital Management Systems
* Library Management Systems
* E-commerce Websites
* Social Media Platforms
