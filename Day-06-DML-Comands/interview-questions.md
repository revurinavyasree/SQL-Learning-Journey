# Day 6 Interview Questions and Answers

## 1. What is DML?

DML stands for Data Manipulation Language.

It is used to manipulate the data stored in database tables.

---

## 2. What are DML Commands?

- INSERT
- SELECT
- UPDATE
- DELETE

---

## 3. What is INSERT?

INSERT is used to add new records into a table.

Example:
Adding a new student into a Student table.

---

## 4. What is SELECT?

SELECT is used to retrieve data from a table.

Example:
Viewing all student records.

---

## 5. What is UPDATE?

UPDATE is used to modify existing records.

Example:
Updating a student's age.

---

## 6. What is DELETE?

DELETE is used to remove records from a table.

Example:
Removing a student record.

---

## 7. Difference Between DELETE and TRUNCATE?

DELETE:
- Removes selected rows
- Can use WHERE clause

TRUNCATE:
- Removes all rows
- Cannot use WHERE clause
- Faster

---

## 8. Difference Between DDL and DML?

DDL:
- Works on structure
- CREATE, ALTER, DROP

DML:
- Works on data
- INSERT, SELECT, UPDATE, DELETE
