# Day 5 Interview Questions and Answers

## 1. What is DDL?

DDL stands for Data Definition Language.

It is used to define and modify database structures.

---

## 2. What are DDL commands?

- CREATE
- ALTER
- DROP
- TRUNCATE

---

## 3. What is CREATE?

CREATE is used to create database objects such as tables and databases.

---

## 4. What is ALTER?

ALTER is used to modify the structure of an existing table.

---

## 5. What is DROP?

DROP removes the table structure and data permanently.

---

## 6. What is TRUNCATE?

TRUNCATE removes all rows from a table while keeping the table structure.

---

## 7. Difference between DELETE and TRUNCATE?

DELETE:
- Removes selected rows.
- Can use WHERE clause.

TRUNCATE:
- Removes all rows.
- Cannot use WHERE clause.
- Faster than DELETE.

---

## 8. Difference between DROP and TRUNCATE?

DROP:
- Removes table and data.

TRUNCATE:
- Removes only data.
- Keeps table structure.
