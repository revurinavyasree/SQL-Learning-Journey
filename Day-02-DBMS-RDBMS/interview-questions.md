# Day 2 Interview Questions and Answers

## 1. What is DBMS?

DBMS is software used to store, manage, and retrieve data.

## 2. What is RDBMS?

RDBMS is a DBMS that stores data in related tables using rows and columns.

## 3. Give examples of DBMS.

File System, XML Database.

## 4. Give examples of RDBMS.

MySQL, PostgreSQL, Oracle Database, Microsoft SQL Server.

## 5. What is the main difference between DBMS and RDBMS?

DBMS stores data without necessarily maintaining relationships, whereas RDBMS stores data in related tables.

## 6. Which is more secure, DBMS or RDBMS?

RDBMS is generally more secure.

## 7. Why is RDBMS preferred?

Because it provides better data integrity, security, and relationship management.
