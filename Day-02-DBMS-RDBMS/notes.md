# Day 2 - DBMS vs RDBMS

## What is DBMS?

DBMS (Database Management System) is software used to store, manage, and retrieve data.

Examples:

* File System
* XML Database

### Advantages of DBMS

* Reduces data redundancy
* Improves data sharing
* Provides data security
* Easier data management

---

## What is RDBMS?

RDBMS (Relational Database Management System) is a type of DBMS that stores data in tables consisting of rows and columns.

Examples:

* MySQL
* PostgreSQL
* Oracle Database
* Microsoft SQL Server

### Advantages of RDBMS

* Supports relationships between tables
* Better data integrity
* Reduced redundancy through normalization
* More secure and scalable

---

## Conclusion

RDBMS is an advanced form of DBMS that stores data in related tables and is widely used in modern applications.
