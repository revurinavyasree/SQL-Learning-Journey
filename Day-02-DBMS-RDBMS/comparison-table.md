# DBMS vs RDBMS

| Feature       | DBMS                       | RDBMS                                 |
| ------------- | -------------------------- | ------------------------------------- |
| Full Form     | Database Management System | Relational Database Management System |
| Storage       | Files or tables            | Related tables                        |
| Relationships | Not mandatory              | Supported                             |
| Redundancy    | Higher                     | Lower                                 |
| Normalization | Limited                    | Supported                             |
| Security      | Basic                      | Better                                |
| Examples      | File System                | MySQL, Oracle, PostgreSQL             |
